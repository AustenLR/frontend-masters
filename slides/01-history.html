jQuery

http://jquery.com

Emerged in 2006 as a simple selector engine with a chainable syntax

Competition with other frameworks over performance propelled jQuery to be (at present) the most used JavaScript library on the web.

When it doubt, http://api.jquery.com/ is your best friend

Underscore

http://underscorejs.org/

The tie on jQuery's tux

A lightweight set of utility functions, that rely on ECMAScript 5 native methods

Backbone Models and Collections "borrow" many underscore methods

No longer a hard dependecy: Lodash is a performance-minded alternate implementation of the Underscore API.

If you've ever used Ruby or Prototype.js, you're already familiar with a majority of these methods.
